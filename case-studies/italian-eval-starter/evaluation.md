# Evaluation criteria

This document defines the criteria used to assess model responses for the Italian evaluation tasks.

## Common criteria (all tasks)

### 1. Clarity
- Is the response easy to read and understand?
- Is the language appropriate for a general audience?

### 2. Factual consistency
- Are the facts consistent with the input?
- Are there hallucinations or invented details?

### 3. Task alignment
- Does the response actually address the task?
- Are all required elements present?

## Task-specific criteria

### Summarization
- Key points preserved
- No irrelevant details added

### Classification
- Correct label assigned
- Reasoning (if present) aligns with the label

### Reasoning
- Logical steps
- Practical and feasible suggestions
